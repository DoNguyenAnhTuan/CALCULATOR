# 🧮 CALCULATOR APP with Tkinter

**Ứng dụng máy tính đơn giản được phát triển bằng Python và thư viện GUI Tkinter.**  
Phù hợp với người mới học lập trình muốn thực hành dự án nhỏ có giao diện người dùng.

---

## 📦 Tính năng

- ➕ Cộng
- ➖ Trừ
- ✖️ Nhân
- ➗ Chia
- 👨‍💻 Giao diện đồ họa (GUI) bằng Tkinter
- 🧪 Xử lý lỗi cơ bản (như chia cho 0)

## ⚙️ Cách sử dụng

### 1. Clone repository:

```bash
git clone https://github.com/DoNguyenAnhTuan/CALCULATOR.git
cd CALCULATOR
````

### 2. Chạy chương trình:

```bash
python calculator.py
```

📌 *Yêu cầu:* Python >= 3.7

---

## 🧠 Kiến thức thực hành trong project

* Giao diện người dùng bằng Tkinter
* Bố cục layout: `grid()`
* Xử lý sự kiện: `Button`, `Entry`, `Command`
* Biến toàn cục và thao tác chuỗi
* Cấu trúc điều kiện và exception handling
